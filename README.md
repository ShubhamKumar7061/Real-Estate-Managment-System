# Real Estate Management System

A simple full-stack **Real Estate Management System** built using **HTML, CSS, JavaScript, Node.js, Express.js, and MySQL**.

## Features

* Buyer and Seller login
* Seller can add, update, and delete properties
* Buyer can view and book properties
* Buyer can send message to seller
* Seller can view messages and reply
* Property search feature
* Image upload support

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MySQL
* Extra: C++ property search module

## Project Files

* `index.html` – property page
* `login.html` – login page
* `messages.html` – chat/messages page
* `server.js` – backend server
* `script.js` – property logic
* `messages.js` – chat logic
* `property_search.cpp` – C++ search program

## Run Project

1. Open MySQL and run your database/table queries
2. Open project folder in VS Code
3. Start backend:

```bash id="6g36ar"
node server.js
```

4. Open `login.html` with Live Server
